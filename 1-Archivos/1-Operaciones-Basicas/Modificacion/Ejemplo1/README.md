# Lectura de un archivo de personas
Utiliza como base el 'Ejemplo 2' de la sección 'Creación'
##### Este ejemplo realiza las siguientes acciones:

1. Utiliza el procedimiento 'agregar_personas' para agregar elementos a un archivo
    1. Abre el archivo que recibe
    2. Lee los datos de una persona
    3. Mientras el dni sea distinto de '', agrega el elemento al final de archivo, vuelve a 2.
    4. Cierra el archivo
